# DormNetworkRecords

GitHub Pages에 배포하는 기숙사 Wi-Fi 속도 기록 페이지입니다.

## 설정

1. `public/app.js`에서 `SUPABASE_URL`, `SUPABASE_ANON_KEY` 값을 입력합니다.
2. Supabase에 `SpeedRecord` 테이블을 만들고 최소한 아래 컬럼을 준비합니다.

```sql
create table "SpeedRecord" (
  id bigint generated by default as identity primary key,
  speed numeric not null,
  created_at timestamptz not null default now()
);
```

## 개발

```bash
npm install
npm run dev
```

## Fast.com 유사 측정

Netflix 테스트 URL 목록은 브라우저에서 직접 가져올 수 없어 Supabase Edge Function으로 프록시합니다.

```bash
supabase functions deploy fast-targets
```

함수가 아직 배포되지 않았거나 실패하면 사이트는 Cloudflare 병렬 다운로드 측정으로 자동 대체됩니다.
